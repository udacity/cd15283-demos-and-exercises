# Exercise 1: Introduction to Forecasting

This exercise is designed to be completed using spreadsheets (like Google Sheets) and BI tools (like Tableau), as outlined in the course curriculum.

## Context

You run a bakery and have two years of daily sales data. Your goal is to use this data to make better decisions about how many loaves of bread to bake each day.

## Data

The sales data is located in `data/bakery_sales.csv`.

## Instructions

### Part 1: Forecasting in Google Sheets

1.  **Import the data:** Upload `data/bakery_sales.csv` into a new Google Sheet.
2.  **Build a Naive Forecast:** In a new column, create a "naive" forecast where each day's predicted sales is simply the value from the previous day.
3.  **Build a Moving Average Forecast:** In another column, calculate a 7-day moving average to forecast sales.
4.  **Build a Linear Trend Forecast:** Use the `TREND()` function to fit a line to your sales data and project it forward.
5.  **Visualize:** Create a line chart that plots the actual sales data along with your three forecasts.

### Part 2: Forecasting in Tableau

1.  **Connect to the data:** Open Tableau and connect to the `data/bakery_sales.csv` file.
2.  **Create a view:** Create a line chart showing sales over time.
3.  **Use the "Show Forecast" feature:** Go to the "Analytics" pane and drag the "Forecast" model onto your view.
4.  **Interpret the results:** Analyze the forecast generated by Tableau. How does it compare to your spreadsheet models? What do the prediction intervals tell you about the uncertainty of the forecast?
