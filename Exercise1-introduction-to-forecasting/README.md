# Exercise 1: Introduction to Forecasting

## Context

You run a bakery. Every morning you decide how many loaves to bake. Too many and you throw away bread at closing. Too few and you turn away customers. You have two years of daily sales data — time to put it to use.

## Exercise

Open `starter/starter.ipynb` and implement three forecasting functions:

1. **Naive forecast** — repeat the last observed value
2. **Moving average forecast** — repeat the trailing window-day average
3. **Linear trend forecast** — fit a trend line and extrapolate

Run the notebook to compare the methods on 14-day forecasts.

## Data

- `data/bakery_sales.csv` — daily loaf sales, January 2023 through December 2024
